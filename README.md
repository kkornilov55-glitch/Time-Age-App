# Time & Age Converter

**Time & Age Converter** — это учебное приложение на Windows Forms, разработанное в рамках лабораторной работы №2 (вариант 14) по дисциплине «Алгоритмизация и программирование». Приложение объединяет три функциональных блока: конвертер времени, калькулятор возраста и визуализацию аналоговых часов.

---

## Функциональность

### 1. Конвертер времени (класс `Time`)
- **Секунды → ЧЧ:ММ:СС** (`SecToTime`) – преобразует введённое количество секунд в строку формата `чч:мм:сс`. Поддерживается диапазон от `00:00:00` до `60:00:00` (максимум 216 000 секунд). Выполняется проверка корректности ввода (целое неотрицательное число).
- **ЧЧ:ММ:СС → секунды** (`TimeToSec`) – по трём значениям (часы, минуты, секунды) вычисляет общее количество секунд. Также проверяет входные данные на целочисленность и неотрицательность.

### 2. Калькулятор возраста (класс `Age`)
- **Расчёт возраста** (`CalcAge`) – принимает дату рождения (день, месяц, год) и возвращает точный возраст в формате *«Вам X лет, Y месяцев, Z дней»*. Учитываются високосные годы, фактическое количество дней в месяцах, а также проверка, не находится ли дата рождения в будущем. Если сегодня день рождения – выводится поздравление.

### 3. Аналоговые часы (реализованы в форме `MainForm`)
- Ввод времени (часы 0–23, минуты 0–59, секунды 0–59).
- При нажатии кнопки «Нарисовать» в элементе `PictureBox` отображается циферблат с часовой, минутной и секундной стрелками.
- Стрелки рассчитываются по углам: секундная – `секунды * 6°`, минутная – `минуты * 6° + секунды * 0.1°`, часовая – `(часы % 12) * 30° + минуты * 0.5°`. Для корректного отображения в системе координат GDI+ используется сдвиг на -90°.
- Часовые метки (12 штук) рисуются с помощью цикла.

### Навигация
- Главное меню содержит кнопки перехода к каждой задаче и справочным панелям.
- Все панели имеют размер 400×500 и переключаются методом `OpenNeedPanel`, который скрывает остальные панели и подгоняет размер формы под активную панель.

## Тестирование

Для классов `Time` и `Age` написаны модульные тесты, покрывающие:
- Корректные преобразования (граничные значения, типовые примеры).
- Обработку ошибок (нечисловой ввод, отрицательные числа, превышение лимитов).
- Специфику дат (високосные года, 29 февраля, месяцы с 30/31 днём).

Все 33 теста успешно проходят, что подтверждает надёжность вычислений.
