# Time & Age Converter


**Time & Age Converter** — это приложение на Windows Forms, которое позволяет:
- Конвертировать секунды в формат `чч:мм:сс` и обратно.
- Вычислять точный возраст (полных лет, месяцев, дней) на основе даты рождения.
- *(В разработке)* Возможность работы с часами (класс `Watch`).

Проект разрабатывается в учебных/демонстрационных целях и сопровождается модульными тестами (MSTest).

---

## Функциональность

### Класс `Time`
- `SecToTime(string sec)` — преобразует строку с секундами в строку формата `чч:мм:сс`.  
  Поддерживается диапазон от `00:00:00` до `60:00:00` (макс. 216 000 секунд). Выполняет корректность ввода.
- `TimeToSec(string hours, string mins, string secs)` — преобразует часы, минуты, секунды в общее количество секунд.  
  Также выполняет проверку корректности ввода.

### Класс `Age`
- `CalcAge(string[] spawnDate)` — принимает массив из трёх строк: `[день, месяц, год]` рождения и возвращает строку с возрастом.  
  Учитывает високосные годы, количество дней в месяцах, проверяет, родился ли человек ещё.  
  Если сегодня день рождения, возвращает поздравление) 

### Класс `Watch`
- Пустой класс — заготовка для будущей функциональности (визуализация времени на часах).

### Главная форма (`MainForm`)
- Обеспечивает навигацию между панелями (меню, конвертер времени, возраст, тестовая панель для `Watch`).
- Перемещает панели непосредственно на форму, чтобы избежать вложенности контейнеров.
